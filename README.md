Paytunia
=

The official Ruby gem to integrate your app with [Paytunia](https://paytunia.com).

## Installation

```
$ sudo gem install paytunia --pre
```

or add it to your `Gemfile`

```ruby
gem 'paytunia', '0.1alpha'
```
